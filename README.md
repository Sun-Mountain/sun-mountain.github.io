## Set Up & Installation

Clone from Github:
```
git clone git@github.com:Sun-Mountain/Sun-Mountain.github.io.git
cd Sun-Mountain.github.io.git
```

Install dependencies:
```
yarn install
```

Run locally:
```
yarn vite
```
